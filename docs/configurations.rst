Configurations
==============

Install
-------

``django-dal`` is available on :samp:`https://pypi.python.org/pypi/django-dal/` install it simply with:

.. code-block:: bash

    $ pip install django-dal

Configure
---------

Settings
~~~~~~~~

Add ``django_dal`` to your ``INSTALLED_APPS``

.. code-block:: python

    INSTALLED_APPS = [
        # ...
        'django_dal',
        # ...
    ]
